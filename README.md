#Insta Clone

Short description:
A mini Instagram clone built with React and Node.js. Users can sign up, log in, post images, like and comment on posts, and interact with other users.

## Features
- User authentication with JWT  
- Create, edit, and delete posts  
- Like and comment on posts  
- Follow and unfollow other users  
- Responsive design for mobile and desktop  
- Real-time notifications (if implemented)  


## Tech Stack
- **Frontend:** React, CSS, React Router  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **Other Tools:** Nodemon for server auto-reload  



## Installation
https://github.com/iamatultiwari/insta-clone.git
